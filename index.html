SCII Guidelines

Solar Conquest 2 Submission Guidelines Written by Zalthen

Welcome to the Submission Guidelines for Solar Conquest 2! Please read these carefully and take them into consideration when submitting a planet or model you want to see in game.

Submission Rules

  1. No Stealing Models from games or other people. If caught, you'll be removed from the community
  2. When criticizing models, be respectful and make your feedback constructive. Avoid needlessly bashing someone's work. Not being a decent person will get you muted.

Planet Guidelines

  • Make your planet terrain unique, look at some popular movie franchises for inspiration!
  • Keep everything on a 1 stud increment. You can use 0.5 increment parts but do so very sparingly!
  • Use rotated parts very sparingly, keep everything at 90 degree increments
  • If you want to add a stud texture to a part with a material not set to plastic, use a texture with the id 3142967175
  • If turning collision off a part, make sure to name the part NoCollide so it's easier for us developers to find it!
  • When adding water to your planet, create a part with the name WaterBlock, it'll be replaced with terrain water!
  • If you want to add air pockets to water, create a part with the name AirBlock, it'll remove the water inside of it!

Model Guidelines

  • Make sure to build with a similar style to what Solar Conquest 2 models currently have, don't make it too detailed and don't make it too simple!
  • If you want a part to have a team color, then name it Color
  • If you want to use a turret another model has, I recommend going ingame and copying the turret!
  • Avoid rotating parts in non-90 degree increments. You can have parts on wedges!
  • When submitting your model, provide some stats for it! Note that we are allowed to change the stats to whatever we deem balanced

Tips and Tricks

! Note that the code snippets below are meant to be ran in the Command Bar

Need to recolor multiple parts with a specific name at once? Use this code snippet below! 

local name = "Leaf"
local color = Color3.fromRGB(35, 80, 23)

for i, obj1 in game.Selection:Get() do
    if obj1:IsA("BasePart") and obj1.Name == name then
        obj2.Color = color
    elseif obj1:IsA("Model") or obj1:IsA("Folder") then
        for j, obj2 in obj1:GetDescendants() do
            if obj2:IsA("BasePart") and obj2.Name == name then
                obj2.Color = color
            end
        end
    end
end

Too many panel gaps in your build and you need to remove them? Use this code snippet below! 

local n = 4
local function roundVector(v)
    return Vector3.new(
        math.round(v.X * n) / n,
        math.round(v.Y * n) / n,
        math.round(v.Z * n) / n
    )
end

local function RoundPartPositionsInContainer(container)
    for _, descendant in ipairs(container:GetDescendants()) do
        if descendant:IsA("BasePart") then
            descendant.Position = roundVector(descendant.Position)
        end
    end
end

for i, obj in game.Selection:Get() do
    if obj:IsA("Model") or obj:IsA("Folder") then
        RoundPartPositionsInContainer(obj)
    elseif obj:IsA("BasePart") then
        obj.Position = roundVector(obj.Position)
    end
end
